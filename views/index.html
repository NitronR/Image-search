Image search

This app can be used to search for images with the query provided by you. The results can be paginated by using a page parameter. The previous searches can also be viewed.

Usage :

To search :
https://image-search-service.glitch.me/search/query_here?page=number_here

To view previous searches:
https://image-search-service.glitch.me/search

Results :

Results will be in JSON format having image URL, title, thumbnail URL and page URL.
Latest queries will also be in JSON format having the search term and the time of search.
